# Limedocs Wiki Converter

> Generate HTML & PDF documentation from you Github wiki or any other markdown-based wiki

## Table of contents

By default, GWC will check for the following files to use as a table of contents (TOC):

- `_Toc.md`
- `_Sidebar.md`
